# 07 - Context dan HOC

![contoh gambar](../../docs/logo/polinema.png)

## Tujuan Pembelajaran

1. Mahasiswa lebih memahami Konsep Context dan manfaatnya.
2. Mahasiswa lebih memahami Konsep HOC di ReactJS.
